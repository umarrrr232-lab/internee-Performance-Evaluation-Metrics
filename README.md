## 📁 Project  — Intern Performance Tracking System

### What it does
Builds a KPI-based scoring system that combines 4 metrics into one performance score out of 100, then visualizes it across departments and months.

### KPI Formula
```
Performance Score =
    Task Completion  × 30% +
    Quality Score    × 25% +
    Mentor Feedback  × 25% +
    Punctuality      × 20%
```

### Performance Labels
| Score | Label |
|-------|-------|
| 80+   | Excellent |
| 65–79 | Good |
| 50–64 | Average |
| <50   | Needs Improvement |

### Charts included
- Performance category distribution
- Avg score by department
- Monthly performance trend
- Quality vs mentor feedback (scatter)
- Task completion spread (box plot)
- Top 10 performers
